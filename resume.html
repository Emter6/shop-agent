基于 LangGraph 构建的电商数据分析智能 Agent，用户以自然语言提问，系统自动完成关键词抽取→多路召回→SQL生成→校验执行→结果返回的完整闭环。采用 多路混合检索策略（Qdrant 向量检索 + Elasticsearch 全文检索 + MySQL 结构化查询）提升 SQL 生成准确率，支持 SSE 流式响应实时展示执行进度。

项目职责
